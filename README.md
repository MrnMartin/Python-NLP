# Python-NLP
Regroupement de mon travail d'apprentissage du Python et des Bases du Natural Language Processing (NLP) et d'autres languages Informatiques.

Plusieurs séries d'exercices progressifs.

## Base Python
- Afficher prénom et âge.
- Additionner 2 nombres entrés par l'utilisateur.
- Vérifier si un nombre est pair ou impair.
- Boucle : Compter jusqu'à 10.
- Créer une fonction qui renvoie le carré d'un nombre.

## Manipulation de texte
- Compter les mots d'un texte.
- Compter les lettres "a" dans un texte.
- Inverser un mot ou une phrase.
- Remplacer un mot dans une phrase.
- Trouver le mot le plus long dans une liste.

## Dictionnaire
- Crée un dictionnaire et afficher une valeur
- Compter les occurrences dans une liste
- Petit jeu , Le Nombre Mystère
